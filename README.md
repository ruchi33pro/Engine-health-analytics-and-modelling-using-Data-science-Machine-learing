# Engine-health-analytics-and-modelling-using-Data-science-Machine-learing
Predictive Maintenance Model development using Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn and XGBoost. Includes EDA, feature engineering and model comparison &amp; optimization &amp; finalization using Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, AdaBoost and Bagging, by evaluating F1, precision &amp; recall
